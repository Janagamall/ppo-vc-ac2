# PPO vs A2C — TinyReproduction
**ECE 570 Course Project | Track 1: TinyReproductions**

Reproduction of the central stability claim from:
> Schulman, J., Wolski, F., Dhariwal, P., Radford, A., & Klimov, O. (2017).
> *Proximal Policy Optimization Algorithms.* arXiv:1707.06347

---

## Project Goal

Provide a minimal empirical evaluation of whether PPO achieves more stable and
higher-reward training than A2C on two classic control tasks (CartPole-v1 and
Acrobot-v1). Three extensions strengthen the evidence over a single-seed baseline:

1. **Multi-seed training** — 3 seeds (0, 1, 2); both model AND environment seeded; mean ± std reported
2. **Two environments** — CartPole-v1 and Acrobot-v1 to probe generalization
3. **Shaded std bands** — smoothing applied per seed before averaging so bands reflect true across-seed variance

## Dependencies

No datasets or pre-trained models are required. CartPole-v1 and Acrobot-v1 are
bundled with Gymnasium and download automatically on first use.

```bash
pip install stable-baselines3 gymnasium matplotlib numpy
```

Tested with:
- Python 3.12
- stable-baselines3 2.8.0
- gymnasium 1.2.3
- numpy 2.0.2
- matplotlib 3.10.0
- torch 2.10.0 (CPU only — no GPU required)

## How to Run

### Option 1 — Google Colab (recommended)
1. Upload `ppo_vs_a2c.ipynb` to Google Colab
2. Run all cells top-to-bottom (Cell 1 installs dependencies automatically)

### Option 2 — Local Jupyter
```bash
pip install stable-baselines3 gymnasium matplotlib numpy jupyter
jupyter notebook ppo_vs_a2c.ipynb
```
If dependencies are already installed, comment out or skip Cell 1.

### Expected Runtime
Approximately 8–10 minutes on a standard CPU.

### Expected Output
Running all cells produces:

1. Per-seed training logs for CartPole-v1 (PPO + A2C, 3 seeds each)
2. Per-seed training logs for Acrobot-v1 (PPO + A2C, 3 seeds each)
3. `learning_curve_cartpole.png` saved to working directory
4. `learning_curve_acrobot.png` saved to working directory
5. Raw rewards saved to `saved_rewards/` for independent verification
6. Final summary table:

```
=================================================================
FINAL RESULTS -- Mean +/- Std over 3 seeds, 20 eval episodes each
=================================================================
Environment      Algorithm   Mean Reward      Std
-----------------------------------------------------------------
CartPole-v1      PPO               500.0      0.0
CartPole-v1      A2C               500.0      0.0
-----------------------------------------------------------------
Acrobot-v1       PPO              -125.5     62.5
Acrobot-v1       A2C              -346.6    188.5
=================================================================
Higher is better for both environments.
```

> **Note:** Both algorithms reach 500 on CartPole-v1 — this task is too easy
> to expose a gap at 50k timesteps. The stability difference is clearly visible
> on Acrobot-v1. Results may vary slightly across machines due to PyTorch
> version differences, but PPO should consistently outperform A2C on Acrobot-v1.

## Reproducibility

- **Model seeding:** `PPO(..., seed=seed)` and `A2C(..., seed=seed)`
- **Environment seeding:** `env.reset(seed=seed)` at the start of each training run
- **Evaluation:** Separate environment instance, `deterministic=True` (greedy policy)
- **Raw rewards:** Saved as `.npy` files in `saved_rewards/` for independent verification
